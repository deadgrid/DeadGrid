DEAD GRID
Survive. Fight. Dominate the Grid.
DEAD GRID is a fast-paced browser-based zombie survival shooter set in a dark cyberpunk world. Fight endless waves of infected enemies, survive as long as possible, and climb the leaderboard in this arcade-inspired action experience.

🎮 Play Now
🌐 Website: deadgrid.fun

📖 About
DEAD GRID combines simple controls with intense survival gameplay. Players must constantly move, dodge incoming threats, and eliminate zombies while chasing higher scores.
Built entirely with HTML5 Canvas, CSS, and Vanilla JavaScript, the game runs directly in modern browsers with no downloads or installation required.
